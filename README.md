## [Documents](https://fozg.github.io/fozg-ui/) 

> React UI kit used for Fozg.net


## Install

```bash
npm install --save fozg-ui
```

## Usage

```jsx
import React, { Component } from 'react'

import {SelectGroup} from 'fozg-ui'

class Example extends Component {
  render () {
    return (
      <SelectGroup>
        ...
      </SelectGroup>
    )
  }
}
```
